  • Round

Round is a function that will round the provided number to the nearest multiple (to) or nearest whole number.

Example

local toThousands=Round(12345678.9,1000) --returns number 12346000
local toWhole=Round(5.2945) --returns number 5

Parameters

Type Name Description
number number The number to round.
number [denominator=1] The multiple to round to.

Returns

Type Description
number The rounded number.
